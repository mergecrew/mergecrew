# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

By participating in this project — opening issues, commenting on PRs, joining Discussions — you agree to uphold the standards described there.

## Reporting

If you experience or witness behavior that violates the Covenant, contact the maintainers privately:

- Email <conduct@mergecrew.dev> (once provisioned)
- Or open a private message via GitHub to a maintainer

Reports will be reviewed promptly and confidentially. The maintainers reserve the right to remove contributions and restrict participation from contributors whose conduct violates the Covenant.

## Scope

This Code applies to all project spaces — the GitHub repository, Discussions, issues, PRs, and any other community channels operated by the project — and to any contributor representing the project externally.
