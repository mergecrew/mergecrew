# UI screenshots

Generated by [`scripts/screenshots`](../../../scripts/screenshots/) against a
running Mergecrew web app. Filenames follow `<route>.<theme>.<viewport>.png`,
for example `06-timeline.dark.desktop.png`.

Regenerate after any UI change:

```bash
pnpm exec playwright install chromium     # one-time
pnpm compose:full                          # or pnpm dev
pnpm screenshots                           # in another shell
```

The route catalog (names, paths, descriptions) lives in
[`scripts/screenshots/routes.ts`](../../../scripts/screenshots/routes.ts).
